# Real-World Data Project: Diabetes Prediction

## Objective

To analyze a real-world healthcare dataset and build a machine learning model to predict diabetes progression based on medical attributes.

## Dataset

Diabetes Dataset (Scikit-learn Built-in Dataset)

## Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

## Project Workflow

### 1. Data Loading

Loaded the diabetes dataset from Scikit-learn and converted it into a Pandas DataFrame.

### 2. Exploratory Data Analysis (EDA)

* Examined dataset structure and statistical summaries.
* Identified relationships between features.
* Visualized correlations using a heatmap.

### 3. Data Visualization

* Correlation Heatmap
* Actual vs Predicted Values Scatter Plot

### 4. Machine Learning Model

* Algorithm: Linear Regression
* Train-Test Split: 80% Training, 20% Testing

### 5. Model Evaluation

* Evaluated model performance using R² Score.
* Compared actual and predicted values.

## Key Findings

* Several medical features showed strong relationships with diabetes progression.
* The correlation heatmap helped identify influential variables.
* Linear Regression successfully predicted disease progression trends.
* The model demonstrated practical use of machine learning in healthcare analytics.

## Conclusion

This project provided hands-on experience in working with a real-world healthcare dataset. It involved data analysis, visualization, predictive modeling, and performance evaluation, helping to strengthen practical data science skills.

## Learning Outcomes

* Real-world data analysis
* Data visualization
* Feature relationship analysis
* Predictive modeling
* Model evaluation using R² Score
* Healthcare data analytics
